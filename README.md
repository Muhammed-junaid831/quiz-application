# Quiz Application (Java Console Project)

## 📌 Project Description
This is a **Java console-based Quiz Application** that allows users to answer multiple-choice questions and calculates their final score based on correct answers.

The project demonstrates basic **Object-Oriented Programming (OOP)** concepts such as classes, objects, arrays, and user input handling.

---

## 🚀 Features
- Displays multiple-choice questions
- Accepts user input from the console
- Stores user answers
- Compares answers with correct answers
- Calculates and displays the final score

---

## 🛠 Technologies Used
- Java
- OOP Concepts
- Arrays
- Scanner (User Input)

---
